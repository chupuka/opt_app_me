## Изменение паролей

Для изменения паролей пользователей можно:
1. Использовать интерфейс администратора (если реализован)
2. Изменить код в `DbInitializer.cs` и перезапустить приложение
3. Использовать инструменты управления пользователями ASP.NET Identity
